# 🚀 KubeaszPull 快速启动指南

## 📋 系统要求
- Node.js 16.0+ 
- npm 8.0+
- 支持的操作系统：Windows、Linux、macOS

## ⚡ 一键启动

### Windows 用户
```bash
# 双击运行
start.bat

# 或在命令行运行
start.bat
```

### Linux/Mac 用户
```bash
# 添加执行权限
chmod +x start.sh

# 运行启动脚本
./start.sh
```

### 手动启动
```bash
# 安装依赖
npm install

# 启动服务器
npm start
# 或
node server.js
```

## 🌐 访问系统

启动成功后，在浏览器中访问：

- **前台页面**：http://localhost:3000
- **后台管理**：http://localhost:3000/admin

## 🔑 默认账号

- **用户名**：admin
- **密码**：admin123

## 📱 功能测试

### 1. 测试付费获取串码
1. 访问前台页面
2. 填写邮箱地址
3. 选择有效期（默认15天）
4. 点击"立即购买"
5. 系统会生成唯一串码

### 2. 测试串码验证
1. 使用生成的串码
2. 在右侧"使用串码查看镜像"区域
3. 输入串码并验证
4. 查看镜像列表

### 3. 测试管理后台
1. 访问 /admin 页面
2. 使用默认账号登录
3. 添加测试镜像
4. 查看串码管理

## 🔧 常见问题

### Q: 启动失败怎么办？
A: 检查Node.js版本，确保版本16.0+

### Q: 端口3000被占用？
A: 修改server.js中的port变量

### Q: 数据库文件在哪？
A: 首次运行会自动创建kubeaszpull.db

### Q: 如何修改默认价格？
A: 在管理后台的系统设置中修改

## 📞 技术支持

如遇问题，请检查：
1. Node.js版本是否正确
2. 依赖是否安装完整
3. 端口是否被占用
4. 防火墙设置
