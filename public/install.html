首次安装

先完成安装配置，再进入 ImgPull 正式页面

首次启动时，系统会在这里写入站点配置、数据库类型和可选的 Harbor 连接信息。当前版本支持 SQLite 和 MySQL 5 到 8 二选一，Harbor 配置可在安装时填写，也可以先跳过，安装完成后再到后台补充。

SQLite 兼容当前仓库已有数据，适合轻量部署和本地测试。
MySQL 已支持安装阶段的连接测试与配置保存，但运行时主逻辑仍以 SQLite 兼容路径为主。
Harbor 建议在安装时先测通地址、账号和密码，后面镜像分发会直接使用这里的配置。
安装前说明
1. 如果你暂时只想把系统跑起来，建议先选 SQLite。
2. 如果你计划后面接真实 Harbor，同步把 Harbor 配置测通会更省事。
3. 首次安装完成后，首页会自动切换成正式前台入口。
