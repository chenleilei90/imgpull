海外公开镜像到国内用户仓库

把 DockerHub 公开镜像同步到你自己的国内仓库

绑定 Harbor、阿里云 ACR 或腾讯云 TCR，输入镜像后由平台执行 pull、tag、push，最终由你从自己的仓库地址拉取。

使用流程

1. 绑定国内仓库
保存 Harbor / ACR / TCR 的目标命名空间与凭证。
2. 输入公开镜像
首发支持 DockerHub 官方公开镜像。
3. 平台执行同步
Worker 执行 pull、tag、login、push。
4. 从你的仓库拉取
复制 docker / nerdctl / crictl 拉取命令。
