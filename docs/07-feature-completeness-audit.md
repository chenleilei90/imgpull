## P0 轻量批量导入

P0 包含轻量批量导入：

- 多行文本粘贴。
- 最多 50 行源镜像。
- 支持解析预览。
- 支持 `source => target` 映射，但多目标项目模式下不支持逐行 targetImage。
- 支持最多 3 个目标项目。
- 单次最多创建 150 条 ImageTask。
- 每个源镜像 + 每个目标项目会创建 1 条独立 ImageTask。
- 每条任务独立冻结、执行、结算、失败返还。
- batchNo 只用于展示和排查，不改变 Worker 执行模型。

## P0 项目 / namespace

P0 包含：

- 用户手动维护项目 / namespace。
- 新建任务时选择已维护项目。
- 项目搜索和多选。
- 项目状态提示。

P0 不包含：

- 自动读取 Harbor project。
- 自动读取云厂商 namespace。
- 自动创建 project / namespace。

## P1 namespace 自动读取

P1 规划：

- Harbor project 自动读取。
- 阿里云 ACR namespace 自动读取。
- 腾讯云 TCR namespace 自动读取。
- 华为云 SWR namespace / organization 自动读取。
- 火山云 CR namespace 自动读取。

OpenAPI 凭据和 Registry 推送凭据必须分开管理。
